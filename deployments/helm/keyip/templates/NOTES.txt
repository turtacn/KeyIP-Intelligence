== Prerequisites ==

Before the application can start, you must ensure the following external
dependencies are available:

  1. PostgreSQL, Neo4j, Redis, OpenSearch, Milvus, Kafka, MinIO, and Keycloak
     services are reachable at the addresses configured in config.yaml.
  2. A Kubernetes Secret named "{{ .Values.secrets.existingSecretName }}" exists
     in namespace "{{ .Values.global.namespace }}" with all required keys
     (postgres-password, neo4j-password, redis-password, opensearch-username,
     opensearch-password, milvus-username, milvus-password, minio-access-key,
     minio-secret-key, keycloak-client-secret, jwt-secret, strategy-gpt-api-key,
     etc.).
  3. (Optional) A ConfigMap named "keyip-env" in namespace
     "{{ .Values.global.namespace }}" for additional environment variables.

== Verifying the deployment ==

  kubectl get pods -n {{ .Values.global.namespace }}
  kubectl get svc -n {{ .Values.global.namespace }}

== Accessing the API ==

  {{- if .Values.ingress.enabled }}
  The API is exposed via ingress at:
    https://{{ .Values.ingress.host }}
  {{- else }}
  Port-forward to the API server:
    kubectl port-forward -n {{ .Values.global.namespace }} svc/{{ include "keyip.fullname" . }}-apiserver 8080:80
  Then access: http://localhost:8080
  {{- end }}
